## 📂 Python 影像辨識小遊戲
此作品為「Python 程式設計」課程所製作的兩個影像辨識小遊戲，透過鏡頭捕捉手部動作，達成互動遊戲體驗。

### 使用工具
* `Python`
* `MediaPipe`：用於即時手部關鍵點追蹤
* `OpenCV`：處理影像輸入、色彩
* `Pygame`：處理遊戲迴圈、碰撞偵測 (用於接蘋果遊戲)
* `cv2.imshow`：利用 OpenCV 視窗設計 UI (用於猜拳遊戲)

### 作品展示
**1. 接蘋果遊戲** (Mediapipe_Pick up the apple_Game.py)
  * 透過偵測玩家的手部位置來控制籃子左右移動，當蘋果碰到籃子時即可得分
  * **操作方式：**：
    1. **移動**：偵測到「左手」籃子向左；偵測到「右手」籃子向右。
    2. **暫停**：做出「握拳」手勢，籃子即停止移動。
   
**1. 剪刀石頭布遊戲** (Mediapipe_rock paper scissors_Game.py)
  * 透過偵測玩家的手勢(剪刀、石頭、布)，與電腦進行對戰
  * **操作方式：**：
    1. 每局倒數 3 秒鐘，於倒數計時前對鏡頭比出「剪刀」、「石頭」或「布」
    2. 系統會自動判讀手勢並決定勝負
